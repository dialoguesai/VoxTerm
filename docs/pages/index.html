VoxTerm

Local, offline voice transcription for your terminal.
Runs on Apple Silicon with MLX. No cloud. No API keys.

curl -fsSL https://dmarzzz.github.io/VoxTerm/install.sh | bash

100% Offline

All processing happens on-device. Your audio never leaves your machine.

Speaker ID

Identifies and remembers speakers across sessions using voice embeddings.

Real-time

Live transcription with Qwen3-ASR on Metal GPU. Sub-second latency.

Terminal Native

Beautiful TUI with cyberpunk theme. FFT waveform visualizer built in.

Multi-language

Supports multiple languages with automatic language detection.

System Audio

Captures both microphone and system audio via ScreenCaptureKit.
